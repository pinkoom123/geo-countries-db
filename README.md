# geo-countries-db
🌍 Comprehensive dataset of ~250 countries with ISO 3166-1 alpha-2/alpha-3 codes, UN M49 region/sub-region codes, dial codes, capitals, currencies, primary languages, land area (km²), nominal GDP (USD), and bounding box coordinates. Sourced from reliable geonames and UN standards for global mapping, data analysis, and geospatial applications.
